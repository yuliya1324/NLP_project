На данной странице представлена интсрукция по использованию корпуса.

В корпусе реализовано несколько видов поисковых запросов:

  1. "кошка" - поиск по конкретной словоформе
  2. кошка - поиск по всем словоформам данной лексемы
  3. NOUN - поиск по части речи
  4. кошка+NOUN - поиск одновременно по лемме и части речи (! слово должно стоять в начальной форме !)

Возможен поиск последовательностей из 1-3 слов.
